**RF Signal Strength Indicator PCB**

**Overview**

This project is a simple RF Signal Strength Indicator designed using KiCad. The circuit detects nearby radio frequency signals using a wire antenna and indicates the presence of RF signals through an LED.

**Components**

- BC547 NPN Transistor
- Resistor (100kΩ)
- Resistor (1kΩ)
- Capacitor (100nF)
- LED
- Wire Antenna
- Power Connector (5V)

**Working**

The antenna receives RF signals from the surrounding environment. The signal is coupled to the base of the transistor through a capacitor. When RF signals are detected, the transistor switches ON and the LED glows, indicating signal presence.
